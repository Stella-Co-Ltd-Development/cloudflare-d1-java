# Project Requirements

This directory contains the high-level project requirements for `cloudflare-d1-java`.

These documents complement the implementation-specific documents in:

```text
docs/implementation-requirements/
```

## Files

| File | Purpose |
|---|---|
| `phase-00-governance.md` | Repository governance and OSS operation requirements |
| `phase-01-product-scope.md` | Product scope, target users, non-goals, and Maven coordinates |
| `phase-02-public-api.md` | Public API requirements |
| `phase-03-http-retry-validation.md` | HTTP, retry, and validation requirements |
| `phase-04-response-errors-mapping.md` | Response model, error, and mapping requirements |
| `phase-05-testing.md` | Testing requirements |
| `phase-06-release-and-maven-central.md` | Release and Maven Central requirements |

## Rule

When implementation-specific details conflict with these high-level requirements, prefer the more specific document under:

```text
docs/implementation-requirements/
```
