# Code of Conduct

## Our Pledge

This project is committed to providing a respectful, welcoming, and constructive environment for contributors and users.

## Expected Behavior

Participants are expected to:

- Be respectful and considerate
- Communicate constructively
- Accept feedback gracefully
- Focus on what is best for the project and community
- Be patient with new contributors
- Respect differing viewpoints and experiences

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, intimidation, or threats
- Discriminatory or offensive comments
- Personal attacks
- Trolling or sustained disruption
- Publishing private information without permission
- Inappropriate sexual language or attention
- Any conduct that maintainers reasonably determine to be harmful to the project community

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that do not align with this Code of Conduct.

Maintainers may also temporarily or permanently restrict participation for behavior they deem inappropriate, harmful, or disruptive.

## Reporting

Please report unacceptable behavior through a private channel provided by the maintainers.

Do not use public issues to report sensitive interpersonal or security matters.

If GitHub private reporting or security advisories are available for this repository, prefer those private channels.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, documentation, and other public or private project-related communication.
